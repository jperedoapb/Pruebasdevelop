# Spring Framework 5: Inicio de Proyecto

Este es un repositorio para ir mostrando poco a poco el desarrollo de un proyecto con SpringBoot.
##hithi¿Qué son los microservicios en Spring boot?
La arquitectura de microservicios es una arquitectura que recientemente, ha ganado mucha popularidad, existiendo casos de éxito reconocidos mundialmente como Netflix o Amazon. Esta arquitectura conlleva un gran esfuerzo, pero para aplicaciones que necesiten ser escalables y muy flexibles es una arquitectura ideal
